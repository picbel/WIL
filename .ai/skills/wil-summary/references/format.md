# WIL Document Structure

This document describes the structure of WIL (Weekly I Learned) files used in this project.

## File Path Pattern
Files are organized by year and month: `YYYY/MM/wil.md` (e.g., `2024/01/wil.md`).

## Front Matter
Each file starts with YAML front matter:
```yaml
---
description: Week I Learnt
keywords: [study, til, lwil, keyword1, keyword2]
---
```

## AI Processing Rules
- Always identify the topic using the `###` headers.
- When summarizing, focus on the core concepts described under the topics.
- When generating quizzes, use the detailed descriptions and code blocks to create questions.
