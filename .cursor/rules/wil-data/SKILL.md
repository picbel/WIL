# WIL Data Skill

이 스킬은 Markdown 문서를 JSON 등 구조화된 데이터로 변환하거나 통계를 제공합니다.

## Command

### `/wil-data`
- **목적**: 문서의 계층 구조를 객체화하여 분석 데이터로 활용.
- **동작**: 주제별 키워드 맵 생성, 날짜별 학습량 통계 등을 산출합니다.

- 파일 구조는 [references/format.md](references/format.md)를 참조하세요.
