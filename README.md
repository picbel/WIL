# 🚀 WIL (Weekly I Learned)

WIL 프로젝트는 한 주간 배운 내용들을 복습하고, 이를 체계적으로 관리하기 위한 개인 학습 메모장입니다. 단순히 적는 것에 그치지 않고, AI 에이전트(Gemini, Claude, Cursor)를 활용하여 학습 효율을 극대화합니다.

## 💡 학습 원칙
이 저장소는 다음 3단계를 통해 학습 내용을 내 것으로 만듭니다:
1. **기록해야 될 키워드를 미리 적습니다**: 기억해야 할 핵심 키워드를 날짜별로 미리 적습니다.
2. **복습 및 서술**: 추후 해당 키워드를 보고 관련 내용을 자신의 언어로 설명하고 기록합니다.
3. **지식 내재화**: 이 과정을 통해 학습한 내용을 다시 한번 복기하며 완전한 내 지식으로 만듭니다.

## 🤖 AI 에이전트 스킬 (Powered by AI)
이 프로젝트는 AI 에이전트 전용 스킬들을 내장하고 있어, `/명령어`를 통해 학습 내용을 스마트하게 관리할 수 있습니다.

| 스킬 | 명령어            | 주요 기능 |
| :--- |:---------------| :--- |
| **요약** | `/wil-summary` | 특정 기간의 학습 내용을 핵심 요약하여 제공합니다. |
| **정제** | `/wil-refine`  | 작성된 내용을 기술적으로 검증하고 문장을 다듬어 최적화합니다. |
| **퀴즈** | `/wil-quiz`    | 학습 내용을 바탕으로 복습용 퀴즈를 생성하여 이해도를 점검합니다. |
| **로드맵** | `/wil-roadmap` | 지금까지의 학습 이력을 분석하여 다음에 학습할 방향을 추천합니다. |
| **데이터화** | `/wil-data`    | Markdown 데이터를 AI가 분석하기 좋은 구조화된 형식으로 변환합니다. |
| **동기화** | `/agent-sync`            | Gemini, Claude, Cursor 등 다양한 에이전트 간의 스킬과 규칙을 동기화합니다. |

## 📁 프로젝트 구조
학습 데이터는 연도 및 월별로 체계적으로 관리됩니다.

```text
.
├── 2023/ - 2026/          # 연도별 학습 기록 폴더
│   └── 01 - 12/           # 월별 폴더
│       └── wil.md         # 실제 학습 기록 파일
├── .ai/skills/            # AI 에이전트용 스킬 정의 (Global)
└── README.md              # 프로젝트 가이드
```

## 🛠️ 사용 방법
1. **기록하기**: `YYYY/MM/wil.md` 파일에 날짜와 키워드를 적습니다. (예: `260315 : Contextual Retrieval 요약하기`)
2. **복습하기**: 작성한 키워드에 대한 상세 내용을 채워 넣습니다.
3. **AI 활용하기**: 지원되는 에이전트(Claude Code, Gemini CLI 등)에서 위 명시된 명령어를 입력하여 학습을 보조받습니다.
